# Resume_Checker
An AI-powered Resume Checker that analyzes resumes for format, content, and keyword optimization to help job seekers improve their chances with ATS and recruiters.
## Features
- Analyze resume structure and formatting
- Highlight missing or weak keywords
- Provide content improvement suggestions
- Supports multiple file formats (PDF, DOCX)

## How It Works
1. Upload your resume.
2. The agent parses the content and evaluates it.
3. Get a detailed report with recommendations to improve your resume.
